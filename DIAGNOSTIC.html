🎯 How Images Work

Image Path Explanation:

Since wine.html is in the public/ folder and images are also in the public/ folder, they use relative paths:

<img src="vlad-zaytsev-sO26ESssFdE-unsplash.jpg" alt="Wine">

This means: "Look for the image in the same folder as this HTML file"

🔧 If Images Don't Show

  1. Hard Refresh: Press Ctrl + Shift + R (Windows) or Cmd + Shift + R (Mac)
  2. Clear Cache: Go to browser settings and clear cached images
  3. Check Console: Press F12 and look for errors in the Console tab
  4. Try Different Browser: Open in Chrome, Firefox, or Edge
  5. Check File Location: Verify images are in public/ folder
