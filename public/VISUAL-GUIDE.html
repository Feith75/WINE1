🚀 How to Start

Easiest Way: Double-click OPEN-SHOP.bat in the main folder

Or manually:

  • Navigate to the public folder
  • Double-click index.html
  • Your browser will open the welcome page
  • Click "Enter Shop" or "Gift Packages"

🔧 If You See "Chinese Characters"

This is a browser cache issue. The files are correct, but your browser is showing old cached content.

  • Solution 1: Press Ctrl + Shift + R (hard refresh)
  • Solution 2: Clear browser cache in settings
  • Solution 3: Open in a different browser
  • Solution 4: Open in incognito/private mode

💡 Quick Tips

  • Backend is only needed for MongoDB storage and M-Pesa payments
  • All prices are in KSh (Kenya Shillings)
